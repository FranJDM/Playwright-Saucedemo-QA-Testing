# SauceDemo Automation Project 🚀

Este proyecto demuestra habilidades de automatización de pruebas E2E utilizando **Playwright** y el patrón **Page Object Model (POM)**.

## 🛠️ Tecnologías
- Framework: Playwright
- Lenguaje: TypeScript
- Patrón: POM (Page Object Model)

## 🧪 Casos de Prueba
- Login exitoso.
- Flujo completo de compra (Agregar al carrito -> Checkout -> Validación).

## 🚀 Cómo ejecutarlo
1. Clonar el repo: `git clone <tu-url>`
2. Instalar dependencias: `npm install`
3. Ejecutar pruebas: `npx playwright test`
4. Ver reporte: `npx playwright show-report`
